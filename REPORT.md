## Задание 1.1: Предикат обработки списка

1. `lasto(List, X)` - получает последний элемент списка (без стандартных предикатов)

Примеры использования:
```prolog
?- lasto([1, 2, 3, 4], X).
X = 4 .
?- lasto([1], X).
X = 1 .
?- lasto([3, 4, 5, -3], X).
X = -3 .
?- lasto([], X).
false.
?- lasto([5, 3, 8, 992], 992).
true .
```

Реализация:
```prolog
lasto([X], X).
lasto([_|T], X) :-
	lasto(T, X).
```

* Если в списке только X элемент, то он и является последним;
* Рекурсивно "отбрасывая" голову списка, проходим по нему. В конце концов T станет списком с одним элементом;
* Список не должен быть пустой.  

2. `lastoo(List, X)` - получает последний элемент списка (со стандартными предикатами)

Примеры использования:
```prolog
?- lastoo([1, 2, 3, 4], X).
X = 4 .
?- lastoo([], X).
false.
?- lastoo([5, 3, 8, 992], 992).
true .
?- lastoo([3, 4, 5, -3], X).
X = -3 .
?- lastoo([92], X).
X = 92 .
```

Реализация:
```prolog
lastoo(Z, X) :- 
	append(_, [X], Z).
```

* Если Z - это конкатенация анонимной переменной и списка, состоящего из одного элемента X, то X - это и есть последний элемент.
